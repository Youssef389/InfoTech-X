Module 2: Data Types & Operators

2.1 Data Types

Basic data types in R include numeric, integer, character, and logical...

# Define variables of different types
num <- 42
int <- 10L
char <- "Hello"
logi <- TRUE

# Print variables
print(num)
print(int)
print(char)
print(logi)

2.2 Operators

Operators in R include arithmetic, relational, logical, and assignment operators...

# Arithmetic operations
a <- 5
b <- 3
sum <- a + b
diff <- a - b
prod <- a * b
quot <- a / b
mod <- a %% b

# Print results
print(paste("Sum:", sum))
print(paste("Difference:", diff))
print(paste("Product:", prod))
print(paste("Quotient:", quot))
print(paste("Modulus:", mod))

Module 3: Control Structures

3.1 Conditional Statements

If-Else statements help in making decisions in R programming...

# Conditional statement
number <- 15

if (number %% 2 == 0) {
    print("The number is even.")
} else {
    print("The number is odd.")
}

3.2 Loops

Loops allow you to execute a block of code multiple times...

# For loop
for (i in 1:5) {
    print(paste("For loop iteration", i))
}

# While loop
j <- 1
while (j <= 5) {
    print(paste("While loop iteration", j))
    j <- j + 1
}

# Repeat loop
k <- 1
repeat {
    print(paste("Repeat loop iteration", k))
    k <- k + 1
    if (k > 5) break
}

Module 4: Functions

4.1 Function Basics

Functions in R help in organizing code into reusable blocks...

# Define a function
greet <- function() {
    print("Hello, World!")
}

# Call the function
greet()

4.2 Function Parameters and Return Values

Functions can accept parameters and return values...

# Function with parameters
add <- function(a, b) {
    return(a + b)
}

# Call the function and print result
result <- add(5, 3)
print(paste("Sum:", result))

Module 5: Data Manipulation

5.1 Working with Data Frames

Data frames are used to store tabular data in R...

# Create a data frame
df <- data.frame(Name = c("Alice", "Bob", "Carol"),
                 Age = c(25, 30, 35))

# Print the data frame
print(df)

5.2 Data Transformation with dplyr

The dplyr package provides tools for data manipulation...

# Load dplyr
library(dplyr)

# Use dplyr functions
df_transformed <- df %>%
    filter(Age > 25) %>%
    mutate(AgeInMonths = Age * 12)

# Print the transformed data frame
print(df_transformed)

Module 6: Data Visualization

6.1 Introduction to ggplot2

ggplot2 is a popular package for creating graphics in R...

# Load ggplot2
library(ggplot2)

# Create a simple plot
ggplot(df, aes(x = Name, y = Age)) +
    geom_bar(stat = "identity") +
    theme_minimal() +
    labs(title = "Age of Individuals", x = "Name", y = "Age")

6.2 Customizing Plots

Customizations can enhance the clarity and presentation of your plots...

# Customize the plot
ggplot(df, aes(x = Name, y = Age, fill = Name)) +
    geom_bar(stat = "identity") +
    theme_light() +
    labs(title = "Customized Plot", x = "Name", y = "Age") +
    scale_fill_brewer(palette = "Set3")

Module 10: Best Practices

10.1 Writing Readable Code

Best practices for writing clean and maintainable R code...

# Example of readable code
calculate_mean <- function(values) {
    mean_value <- mean(values)
    return(mean_value)
}

data <- c(1, 2, 3, 4, 5)
mean_result <- calculate_mean(data)
print(paste("Mean:", mean_result))

10.2 Documenting Code

Documenting your code for better understanding and maintenance...

# Documenting code with comments
# This function calculates the sum of two numbers
sum_numbers <- function(a, b) {
    return(a + b)
}

result <- sum_numbers(10, 5)
print(paste("Sum:", result))
