Module 3: Control Structures

3.1 Conditional Statements

Conditional statements in Python include if, elif, and else...

# Conditional example
number = 15

if number % 2 == 0:
    print("The number is even.")
else:
    print("The number is odd.")

3.2 Loops

Python supports for and while loops for iterating over sequences...

# Loop examples
# For loop
for i in range(5):
    print("For loop iteration", i)

# While loop
j = 0
while j < 5:
    print("While loop iteration", j)
    j += 1

Module 10: Object-Oriented Programming

10.1 Classes and Objects

Introduction to classes and objects...

# Class example
class Dog:
    def __init__(self, name):
        self.name = name

    def bark(self):
        return "Woof!"

my_dog = Dog("Buddy")
print(my_dog.bark())

10.2 Inheritance and Polymorphism

Using inheritance and polymorphism to create complex systems...

# Inheritance example
class Animal:
    def speak(self):
        return "Animal sound"

class Cat(Animal):
    def speak(self):
        return "Meow"

my_cat = Cat()
print(my_cat.speak())
