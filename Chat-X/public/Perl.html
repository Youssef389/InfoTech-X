Module 1: Introduction to Perl

1.1 Overview of Perl

Perl is a high-level, general-purpose programming language known for its text-processing capabilities...

#!/usr/bin/perl

print "Hello, World!\n";

1.2 History and Evolution

Perl was developed by Larry Wall in 1987 as a Unix scripting language to make report processing easier...

Module 2: Basic Syntax and Data Types

2.1 Variables

Perl has three types of variables: scalars, arrays, and hashes...

#!/usr/bin/perl

my $scalar = "Hello, World!";
my @array = (1, 2, 3, 4, 5);
my %hash = (key1 => 'value1', key2 => 'value2');

print "$scalar\n";
print "$array[0]\n";
print "$hash{key1}\n";

2.2 Operators

Operators in Perl include arithmetic, relational, and logical operators...

#!/usr/bin/perl

my $a = 10;
my $b = 20;
print "$a + $b = ", $a + $b, "\n";

Module 3: Subroutines and Functions

3.1 Defining and Calling Subroutines

Subroutines in Perl are defined using the `sub` keyword...

#!/usr/bin/perl

sub greet {
    print "Hello, World!\n";
}

greet();

3.2 Parameters and Return Values

Subroutines can accept parameters and return values...

#!/usr/bin/perl

sub add {
    my ($a, $b) = @_;
    return $a + $b;
}

my $result = add(5, 3);
print "Sum: $result\n";

Module 4: File Handling

4.1 Reading from Files

Files can be read using the `open` and `readline` functions...

#!/usr/bin/perl

open(my $fh, '<', 'file.txt') or die "Cannot open file: $!";
while (my $line = <$fh>) {
    print $line;
}
close($fh);

4.2 Writing to Files

Files can be written using the `open` and `print` functions...

#!/usr/bin/perl

open(my $fh, '>', 'file.txt') or die "Cannot open file: $!";
print $fh "Hello, World!\n";
close($fh);

Module 5: Regular Expressions

5.1 Basics of Regex

Regular expressions are used for pattern matching...

#!/usr/bin/perl

my $text = "Hello, World!";
if ($text =~ /World/) {
    print "Pattern found!\n";
}

5.2 Substitution and Translation

Regular expressions can also be used for substitution and translation...

#!/usr/bin/perl

$text =~ s/World/Perl/;
print "$text\n";

Module 7: Error Handling

7.1 Error Handling with `eval`

Errors can be handled using the `eval` block...

#!/usr/bin/perl

eval {
    my $result = 1 / 0;
};
if ($@) {
    print "Error: $@\n";
}

7.2 Using `die` and `warn`

The `die` and `warn` functions provide error messages...

#!/usr/bin/perl

open(my $fh, '<', 'file.txt') or die "Cannot open file: $!";

Module 8: Object-Oriented Programming

8.1 Introduction to OOP in Perl

Perl supports object-oriented programming...

package Person;

sub new {
    my $class = shift;
    my $self = {
        name => shift,
    };
    bless $self, $class;
    return $self;
}

sub greet {
    my $self = shift;
    print "Hello, ", $self->{name}, "!\n";
}

1;

8.2 Creating Classes and Objects

Classes and objects are created using the `bless` function...

#!/usr/bin/perl

use Person;

my $person = Person->new("Alice");
$person->greet();

Module 9: Advanced Topics

9.1 References

References allow for complex data structures...

#!/usr/bin/perl

my $array_ref = [1, 2, 3];
print $array_ref->[0], "\n";

9.2 File and Directory Manipulation

Perl provides functions to manipulate files and directories...

#!/usr/bin/perl

use File::Spec;

my $file = File::Spec->catfile('path', 'to', 'file.txt');
print "File path: $file\n";

9.3 Network Programming

Perl supports network programming...

#!/usr/bin/perl

use IO::Socket::INET;

my $socket = new IO::Socket::INET(
    PeerAddr => 'www.example.com',
    PeerPort => '80',
    Proto    => 'tcp',
) or die "Cannot connect to server: $!\n";

print $socket "GET / HTTP/1.0\r\n\r\n";
while (my $line = <$socket>) {
    print $line;
}
