Module 1: Introduction to PHP

1.1 Overview of PHP

PHP (Hypertext Preprocessor) is a widely-used open-source scripting language designed for web development...

<?php
echo "Hello, World!";
?>

1.2 Setting Up PHP

To start using PHP, you need to have a server setup like XAMPP or WAMP, and PHP installed...

Module 2: Variables & Data Types

2.1 Variables

Variables in PHP are represented with a dollar sign followed by the variable name...

<?php
$age = 30;
$name = "John";
echo "Name: $name, Age: $age";
?>

2.2 Data Types

PHP supports various data types including integers, floats, strings, arrays, and objects...

<?php
$int = 10;
$float = 10.5;
$string = "Hello, World!";
$boolean = true;

echo "Integer: $int\\n";
echo "Float: $float\\n";
echo "String: $string\\n";
echo "Boolean: " . ($boolean ? "true" : "false") . "\\n";
?>

Module 3: Control Structures

3.1 Conditional Statements

Conditional statements such as if, else, and switch allow you to execute different code based on conditions...

<?php
$number = 15;

if ($number % 2 == 0) {
    echo "The number is even.\\n";
} else {
    echo "The number is odd.\\n";
}
?>

3.2 Loops

Loops allow you to execute a block of code multiple times...

<?php
// For loop
for ($i = 0; $i < 5; $i++) {
    echo "For loop iteration $i\\n";
}

// While loop
$j = 0;
while ($j < 5) {
    echo "While loop iteration $j\\n";
    $j++;
}

// Do-While loop
$k = 0;
do {
    echo "Do-While loop iteration $k\\n";
    $k++;
} while ($k < 5);
?>

Module 4: Functions

4.1 Function Basics

Functions in PHP help in organizing code into reusable blocks...

<?php
function greet() {
    echo "Hello, World!\\n";
}

greet();
?>

4.2 Function Parameters and Return Values

Functions can accept parameters and return values...

<?php
function add($a, $b) {
    return $a + $b;
}

$result = add(5, 3);
echo "Sum: $result\\n";
?>

Module 5: Arrays

5.1 Indexed Arrays

Indexed arrays use numerical indexes to store values...

<?php
$fruits = array("Apple", "Banana", "Cherry");
echo $fruits[1]; // Outputs: Banana
?>

5.2 Associative Arrays

Associative arrays use named keys to store values...

<?php
$ages = array("John" => 30, "Jane" => 25);
echo $ages["Jane"]; // Outputs: 25
?>

5.3 Multidimensional Arrays

Multidimensional arrays contain other arrays...

<?php
$matrix = array(
    array(1, 2, 3),
    array(4, 5, 6),
    array(7, 8, 9)
);
echo $matrix[1][2]; // Outputs: 6
?>

Module 6: Forms & User Input

6.1 Handling Form Data

PHP handles form data via the $_POST and $_GET superglobals...

<form method="post" action="process.php">
    Name: <input type="text" name="name">
    <input type="submit" value="Submit">

<?php
if ($_SERVER["REQUEST_METHOD"] == "POST") {

?>

Module 7: File Handling

7.1 Reading Files

PHP can read files using functions like fopen, fread, and file_get_contents...

<?php
$file = fopen("example.txt", "r");
echo fread($file, filesize("example.txt"));
fclose($file);
?>

7.2 Writing to Files

PHP can write data to files using fwrite and file_put_contents...

<?php
$file = fopen("example.txt", "w");
fwrite($file, "Hello, File!");
fclose($file);
?>

Module 8: Sessions & Cookies

8.1 Sessions

Sessions are used to store data across multiple pages...

<?php
session_start();
$_SESSION["username"] = "JohnDoe";
echo "Username is " . $_SESSION["username"];
?>

8.2 Cookies

Cookies store data on the client side...

<?php
setcookie("user", "JohnDoe", time() + 3600);
if (isset($_COOKIE["user"])) {
    echo "User is " . $_COOKIE["user"];
}
?>

Module 9: Object-Oriented PHP

9.1 Classes and Objects

Object-oriented programming in PHP involves creating classes and objects...

<?php
class Car {
    public $model;
    
    function __construct($model) {
        $this->model = $model;
    }

    function getModel() {
        return $this->model;
    }
}

$myCar = new Car("Toyota");
echo $myCar->getModel();
?>

9.2 Inheritance

Inheritance allows a class to use methods and properties of another class...

<?php
class Vehicle {
    public $brand;
    
    function __construct($brand) {
        $this->brand = $brand;
    }
}

class Car extends Vehicle {
    public $model;

    function __construct($brand, $model) {
        parent::__construct($brand);
        $this->model = $model;
    }

    function getDetails() {
        return $this->brand . " " . $this->model;
    }
}

$myCar = new Car("Toyota", "Corolla");
echo $myCar->getDetails();
?>

Module 10: Error Handling

10.1 Error Reporting

PHP provides error reporting functionality to help identify issues...

<?php
error_reporting(E_ALL);
ini_set("display_errors", 1);
echo $undefinedVariable;
?>

10.2 Exception Handling

Exceptions in PHP help handle errors more gracefully...

<?php
try {
    if (!file_exists("example.txt")) {
        throw new Exception("File not found.");
    }
} catch (Exception $e) {
    echo 'Caught exception: ',  $e->getMessage(), "\n";
}
?>
