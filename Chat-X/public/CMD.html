Introduction to CMD

The Command Prompt (CMD) is a command-line interpreter application available in most Windows operating systems...

Basic CMD Commands

1.1 dir

Lists the files and directories in the current directory.

dir

1.2 cd

Changes the current directory to the specified directory.

cd [directory]

1.3 mkdir

Creates a new directory.

mkdir [directory_name]

1.4 rmdir

Removes a directory.

rmdir [directory_name]

File Operations

2.1 copy

Copies one or more files from one location to another.

copy [source] [destination]

2.2 del

Deletes one or more files.

del [file_name]

2.3 move

Moves one or more files from one location to another.

move [source] [destination]

Networking Commands

3.1 ipconfig

Displays IP address, subnet mask, and default gateway for all network adapters.

ipconfig

3.2 ping

Sends ICMP Echo Request messages to network hosts.

ping [hostname]

3.3 tracert

Traces the route taken by packets to a destination.

tracert [hostname]

Advanced CMD Commands

4.1 tasklist

Displays a list of currently running processes on the local machine.

tasklist

4.2 sfc /scannow

Scans and repairs system files.

sfc /scannow

4.3 chkdsk

Checks the file system and file system metadata of a volume for logical and physical errors.

chkdsk [drive:] [parameters]

CMD Cheat Sheet

Here's a cheat sheet of some common CMD commands:


                ## CMD Commands Cheat Sheet
                
                ### Basic Commands
                - **`--help`**: Displays help information about a command.
                - **`--version`**: Shows the version of the command or program.
                - **`explorer .`**: Opens the current directory in Windows File Explorer.
                - **`cd`**: Changes the current directory.
                - **`mkdir [folder_name]`**: Creates a new directory. (`mkdir " "` creates a directory with a space in the name).
                - **`rmdir [folder_name]`**: Removes a directory.
                - **`code .`**: Opens the current directory in Visual Studio Code.
                - **`clear`**: Clears the terminal screen (use `cls` in Windows).
                - **`ls`**: Lists directory contents (use `dir` in Windows).
                - **`dir`**: Lists the contents of a directory in Windows.
                
                ### File and Directory Operations
                - **`mkdir ../folder_name`**: Creates a new directory in the parent directory.
                - **`mkdir c:\folder_name`**: Creates a directory on the C: drive.
                - **`cd c:\`**: Changes to the C: drive root directory.
                - **`move [moved_folder] [new_folder]`**: Moves a folder to a new location (Windows only).
                - **`mv [moved_folder] [new_folder]`**: Moves or renames a folder (Unix-like systems).
                - **`mv [moved_folder] ..\`**: Moves a folder to the parent directory.
                - **`cp [source] [destination]`**: Copies files.
                - **`cp -r [source] [destination]`**: Recursively copies directories and their contents.
                - **`rm [file_name]`**: Removes a file.
                - **`rm -r [directory_name]`**: Recursively removes a directory and its contents.
                
                ### File and Text Operations
                - **`echo [text] > [file_name]`**: Writes text to a file (creates the file if it doesn‚Äôt exist).
                - **`echo [text] >> [file_name]`**: Appends text to a file.
                - **`cat [file_name]`**: Displays the contents of a file (Unix-like systems).
                - **`cat [file1] [file2] > [new_file]`**: Concatenates files into a new file.
                - **`grep "[pattern]" [file_name]`**: Searches for a pattern in a file (Unix-like systems).
                - **`grep -r -l "[pattern]" [directory]`**: Recursively searches for a pattern in files within a directory.
                
                ### System and Process Management
                - **`exit`**: Exits the command prompt or terminal.
                - **`file [file_name]`**: Displays file type information (Unix-like systems).
                - **`rfile [file_name]`**: Removes a file (Unix-like systems).
                - **`touch [file_name]`**: Creates an empty file or updates the timestamp of a file (Unix-like systems).
                - **`rename [old_name] [new_name]`**: Renames a file or directory.
                - **`man [command]`**: Displays the manual for a command (Unix-like systems).
                - **`/?`**: Displays help information for a command (Windows).
                - **`tree /a`**: Displays a directory tree structure.
                
                ### Additional Commands
                - **`control + c`**: Interrupts a running command or process (Mac only).
                - **`osk`**: Opens the On-Screen Keyboard.
                - **`tasklist`**: Lists all running processes (search for killing processes in the tasklist).
                - **`alias`**: Creates or lists aliases (Unix-like systems).
                - **`ipconfig`**: Displays IP configuration information.
                - **`whoami`**: Displays the current user‚Äôs username.
                - **`systeminfo`**: Displays detailed system information.
                - **`ping [address]`**: Tests network connectivity to an address.
                - **`[command] > [file_name].txt`**: Redirects command output to a file.
                - **`[command] | clip`**: Copies command output to the clipboard (Windows).
