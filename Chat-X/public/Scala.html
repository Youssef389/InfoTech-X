Module 1: Introduction to Scala

1.1 Overview of Scala

Scala is a high-level programming language that combines object-oriented and functional programming principles...

object HelloWorld {
    def main(args: Array[String]): Unit = {
        println("Hello, World!")
    }
}

1.2 History of Scala

Scala was created by Martin Odersky and first released in 2003. It is designed to be concise, elegant, and type-safe...

Module 2: Basic Syntax

2.1 Variables and Data Types

Scala uses both mutable and immutable variables. The val keyword defines an immutable variable, while var defines a mutable one...

object BasicSyntax {
    def main(args: Array[String]): Unit = {
        val name: String = "John Doe"
        var age: Int = 30

        println(s"Name: $name")
        println(s"Age: $age")
    }
}

2.2 Expressions and Statements

Expressions in Scala produce a value, while statements perform an action...

object Expressions {
    def main(args: Array[String]): Unit = {
        val sum = 5 + 10
        println(s"Sum: $sum")
    }
}

Module 3: Data Types

3.1 Primitive Data Types

Scala has several built-in data types such as Int, Double, Boolean, and Char...

object DataTypes {
    def main(args: Array[String]): Unit = {
        val intValue: Int = 42
        val doubleValue: Double = 3.14
        val boolValue: Boolean = true
        val charValue: Char = 'A'

        println(s"Int: $intValue, Double: $doubleValue, Boolean: $boolValue, Char: $charValue")
    }
}

3.2 String Handling

Strings in Scala are immutable and offer various methods for manipulation...

object StringHandling {
    def main(args: Array[String]): Unit = {
        val str = "Hello, Scala!"
        println(s"Length of the string: ${str.length}")
        println(s"Substring: ${str.substring(7)}")
    }
}

Module 4: Control Structures

4.1 Conditional Statements

Scala provides if and match expressions for conditional logic...

object ConditionalStatements {
    def main(args: Array[String]): Unit = {
        val number = 10
        val result = if (number % 2 == 0) "Even" else "Odd"
        println(s"Number is: $result")
    }
}

4.2 Loops

Scala offers for, while, and do-while loops...

object Loops {
    def main(args: Array[String]): Unit = {
        for (i <- 1 to 5) {
            println(s"For loop iteration $i")
        }

        var j = 1
        while (j <= 5) {
            println(s"While loop iteration $j")
            j += 1
        }
    }
}

Module 5: Functions

5.1 Defining Functions

Functions in Scala can be defined using the def keyword...

object Functions {
    def greet(name: String): String = {
        s"Hello, $name!"
    }

    def main(args: Array[String]): Unit = {
        println(greet("Scala"))
    }
}

5.2 Higher-Order Functions

Scala supports higher-order functions, which can take other functions as parameters...

object HigherOrderFunctions {
    def applyFunction(f: Int => Int, x: Int): Int = f(x)

    def main(args: Array[String]): Unit = {
        val result = applyFunction(x => x * x, 5)
        println(s"Result: $result")
    }
}

Module 6: Collections

6.1 Lists

Lists in Scala are immutable and ordered collections of elements...

object Lists {
    def main(args: Array[String]): Unit = {
        val list = List(1, 2, 3, 4, 5)
        println(s"List: $list")
        println(s"Head: ${list.head}, Tail: ${list.tail}")
    }
}

6.2 Maps

Maps are collections of key-value pairs...

object Maps {
    def main(args: Array[String]): Unit = {
        val map = Map("a" -> 1, "b" -> 2, "c" -> 3)
        println(s"Map: $map")
        println(s"Value for key 'b': ${map("b")}")
    }
}

Module 7: Object-Oriented Programming

7.1 Classes and Objects

Scala is a fully object-oriented language where everything is an object...

class Person(val name: String, val age: Int) {
    def greet(): String = s"Hello, my name is $name and I am $age years old."
}

object Main {
    def main(args: Array[String]): Unit = {
        val person = new Person("John", 30)
        println(person.greet())
    }
}

7.2 Inheritance

Inheritance in Scala allows for the creation of new classes based on existing ones...

class Animal {
    def speak(): String = "Animal sound"
}

class Dog extends Animal {
    override def speak(): String = "Woof!"
}

object Main {
    def main(args: Array[String]): Unit = {
        val dog = new Dog
        println(dog.speak())
    }
}

Module 8: Pattern Matching

8.1 Basic Pattern Matching

Pattern matching is a powerful feature in Scala...

object PatternMatching {
    def matchTest(x: Any): String = x match {
        case 1 => "One"
        case "hello" => "Hello"
        case _ => "Something else"
    }

    def main(args: Array[String]): Unit = {
        println(matchTest(1))
        println(matchTest("hello"))
        println(matchTest(3.14))
    }
}

Module 9: File I/O

9.1 Reading Files

Scala can handle file I/O using the scala.io.Source class...

import scala.io.Source

object FileIO {
    def main(args: Array[String]): Unit = {
        val filename = "example.txt"
        val source = Source.fromFile(filename)
        for (line <- source.getLines()) {
            println(line)
        }
        source.close()
    }
}

9.2 Writing Files

Writing to files is also straightforward in Scala...

import java.io.PrintWriter

object FileWrite {
    def main(args: Array[String]): Unit = {
        val writer = new PrintWriter("output.txt")
        writer.write("Hello, Scala!")
        writer.close()
    }
}

Module 10: Concurrency

10.1 Threads

Scala supports concurrency using threads...

object Concurrency {
    def main(args: Array[String]): Unit = {
        val thread = new Thread(new Runnable {
            def run(): Unit = {
                println("Running in a thread")
            }
        })
        thread.start()
        thread.join()
    }
}

10.2 Futures

Futures provide a way to perform asynchronous operations...

import scala.concurrent.Future
import scala.concurrent.ExecutionContext.Implicits.global

object FuturesExample {
    def main(args: Array[String]): Unit = {
        val future = Future {
            Thread.sleep(1000)
            "Hello from Future"
        }

        future.onComplete {
            case Success(value) => println(value)
            case Failure(exception) => println(s"Error: $exception")
        }
    }
}
