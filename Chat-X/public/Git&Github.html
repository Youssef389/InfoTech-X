1.2 Why Git?

Module 2: Git Basics

2.1 Installing Git

Git can be installed on various platforms. Visit the official Git website to download and install...

$ sudo apt-get install git

2.2 Initializing a Repository

Initialize a new Git repository with the git init command...

$ git init my-repo
Initialized empty Git repository in /path/to/my-repo/.git/

Module 3: Branching

3.1 Understanding Branches

Branches allow you to work on different versions of your project simultaneously...

$ git branch feature-branch
$ git checkout feature-branch
Switched to branch 'feature-branch'

3.2 Creating and Switching Branches

Creating and switching between branches is a fundamental skill in Git...

$ git checkout -b new-feature
Switched to a new branch 'new-feature'

Module 4: Merging

4.1 Fast-Forward Merge

A fast-forward merge happens when there is a direct path from the current branch to the target branch...

$ git merge feature-branch
Updating 3b2d3f0..4e6e2a2
Fast-forward
 file.txt | 1 +
 1 file changed, 1 insertion(+)

4.2 Merge Conflicts

Merge conflicts occur when Git cannot automatically resolve differences between two commits...

$ git merge feature-branch
Auto-merging file.txt
CONFLICT (content): Merge conflict in file.txt
Automatic merge failed; fix conflicts and then commit the result.

Module 5: Remote Repositories

5.1 Cloning a Repository

Clone a remote repository to your local machine using the git clone command...

$ git clone https://github.com/user/repo.git
Cloning into 'repo'...
remote: Enumerating objects: 10, done.
remote: Counting objects: 100% (10/10), done.
remote: Compressing objects: 100% (5/5), done.
remote: Total 10 (delta 2), reused 10 (delta 2), pack-reused 0
Unpacking objects: 100% (10/10), done.

5.2 Fetching and Pulling

The git fetch command downloads changes from a remote repository...

$ git fetch origin
$ git pull origin main
